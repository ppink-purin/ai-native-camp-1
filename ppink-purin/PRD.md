# AI 기반 개인 학습 & 생산성 시스템

## 문제
> 온라인 콘텐츠는 많지만 제대로 소화하고 활용하는 사람은 드물다

- **현재 상태**: LinkedIn 피드 100+ 글/일, YouTube 영상 시청 2시간/일, 트윗 스크롤 30분/일 → 실제 기억에 남는 것은 5% 미만. 개발 세션 후 "뭘 했더라?" 되새김에 10분 소요.
- **원하는 상태**: AI가 중요한 콘텐츠를 선별하고, Quiz-First로 기억력 9-12% 향상시키며, 세션마다 자동으로 학습 내용이 문서화되는 시스템
- **성공 기준**:
  - 주 5회 이상 스킬 사용
  - 콘텐츠 소화 시간 50% 단축 (영상 30분 → 15분)
  - 학습 내용 복습 시 기억률 70% 이상

## 스킬
| # | 스킬명 | 한 줄 설명 | 상태 |
|---|--------|-----------|------|
| 1 | `/my-context-sync` | LinkedIn 피드 → AI 분석 → TOP 10 추천 | ✅ 동작 |
| 2 | `/my-fetch-tweet` | X/Twitter URL → 요약/인사이트/전체 번역 | ✅ 동작 |
| 3 | `/my-fetch-youtube` | YouTube URL → 자막 추출 → Web Search 보정 → 번역 | ✅ 동작 |
| 4 | `/my-content-digest` | 콘텐츠 → Pre-Quiz → 선택적 학습 → 본 퀴즈 (기억력 9-12% ↑) | ✅ 동작 |
| 5 | `/my-session-wrap` | Git 변경 → 4개 병렬 에이전트 분석 → 문서 업데이트/커밋 자동화 | ✅ 동작 |

## 변화 기록
- **Day 1**: "Claude Code가 뭐지?" → 기본 개념 이해
- **Day 2**: "정보가 너무 흩어져 있다" → context-sync로 LinkedIn 자동 분석
- **Day 4**: "세션 끝날 때마다 정리가 번거롭다" → session-wrap으로 4개 에이전트 자동 분석
- **Day 5**: "영어 콘텐츠 읽기 부담스럽고 기억에 안 남는다" → fetch (트윗/YouTube) + digest (Quiz-First)
- **Day 6**: "5개 스킬이 하나의 시스템이 되었다" → PRD 작성 & 제출
- **가장 크게 달라진 점**:
  - 콘텐츠를 "읽기만" → "fetch하고 quiz로 내 것으로 만들기"
  - 세션 정리를 "수동" → "4개 에이전트가 자동으로"
  - 흩어진 스킬 → "fetch → digest → wrap" 파이프라인
